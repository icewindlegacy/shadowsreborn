AUTODAMAGE

****************************

Syntax: autodamage

Toggles the automatic display of damage dealt in combat. When enabled,
you will see exact damage numbers during fights. When disabled, you'll
only see the standard damage messages (scratches, bruises, etc.).

This is useful for players who want detailed combat information without
needing to manually check after each round.

AUTOTICK

Toggles automatic notification when game ticks occur. When enabled, you

BANDAGE

****************************

Syntax: bandage <target>

Attempt to bandage yourself or another character to stop bleeding and restore
some health. This skill requires you to be at rest or better, and costs movement

Successful bandaging will:

0 'COMM STONES' COMMSTONES COMMSTONE FREQUENCIES~

COMM STONES

****************************

Syntax: <frequency> <message>
        <frequency> emote <action>
        freq <channel> <frequency>
        freq <channel> off

Commstones are magical devices that allow long-distance communication on
specific frequencies. You can have up to 4 commstone channels active at
once.

TRANSMIT: Simply type the frequency number followed by your message:
  Example: 1 Hello everyone on frequency 0!

EMOTE: You can also send emotes through commstones:
  Example: 1 emote waves enthusiastically

FREQ: Configure your commstone channels:
  freq 1 42    - Set channel 1 to frequency 42
  freq 1 off   - Turn off channel 1

Each character can tune their commstones to different frequencies (0-99999),
allowing for private group communication. Drunk speech and common language
effects apply to commstone communications!

Keyrings are special containers that store keys without taking up

52 PLOAD PUNLOAD~

PLOAD / PUNLOAD

****************************

Syntax: pload <player>
        punload <player>

(Immortal Command - Level 52+)

PLOAD - Load an offline player's character data into memory for editing
or inspection. This allows immortals to modify offline characters or
investigate issues without requiring the player to be online.

PUNLOAD - Unload a player that was loaded with PLOAD, saving any changes
made while they were loaded.

WARNING: Be extremely careful when using these commands! Modifying offline
characters can cause serious problems if not done correctly. Always:
- Note what changes you make
- Save the character after modifications
- Unload the character when finished
- Never leave characters loaded indefinitely

These commands are essential tools for customer service and administration.
